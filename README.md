# Creusot Devlog

https://creusot-rs.github.io/devlog

## How to build

1. Clone this repository and set up submodules

    ```
    git clone --recurse-submodules https://github.com/creusot-rs/devlog
    ```

2. [Install Zola](https://github.com/getzola/zola/releases) (one binary)

3. Build and preview

    ```
    zola serve
    ```
