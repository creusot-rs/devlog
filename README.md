# Creusot Devlog

https://creusot-rs.github.io/devlog

## How to build

1. Clone this repository and set up submodules

    ```
    git clone --recurse-submodules https://github.com/creusot-rs/devlog
    ```

2. [Install Zola](https://github.com/getzola/zola/releases) ([direct link to the 0.22.0 binary for x86\_64 Linux](https://github.com/getzola/zola/releases/download/v0.22.0/zola-v0.22.0-x86_64-unknown-linux-gnu.tar.gz))

3. Build and preview

    ```
    zola serve
    ```
